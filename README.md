# Caveduck 卡片小工具

幾個給 [Caveduck（鴨答）](https://caveduck.io) 角色卡創作者用的直覺化調整小工具

全部都是**單一 HTML 檔、純前端、純本地運作**——打開就能用，不用安裝、不用註冊。

## 🎯 內容對齊工具（aligner_pro.html）

當你有一個「素材框」（掌機、相框、螢幕外框等圖片），卻一直沒辦法讓 CSS 內容剛好對齊框裡的螢幕時，用這個。

**用法**
1. 上傳你的素材圖
2. 貼上要對齊的 HTML 內容（只貼螢幕裡要顯示的部分，例如指令卡 `<div>`）
3. 拖四個角對齊框裡的螢幕
4. 產生程式碼，複製貼回 Caveduck

**特色**
- 拖四角即時預覽，所見即所得
- 輸出可選「清晰版」（文字框正放、字不糊，預設）或「3D 透視版」（完全貼合斜螢幕）
- 產生的程式碼符合 Caveduck 規則（全 inline、無多餘空行）

## 🖱️ 點選微調工具（picker.html）

貼上 AI 或自己寫好的 HTML，直接在預覽裡**點元素就地調整**，不用回頭翻程式碼找參數。

**用法**
1. 貼上 HTML，按渲染
2. 在預覽裡點你想改的元素
3. 元素旁浮出小面板，直接調
4. 產生程式碼，複製貼回 Caveduck

**能調什麼**
- 文字內容（就地打字編輯）
- 反白選字單獨改顏色（像 Word）
- 文字色 / 背景色 / 邊框色
- 字級、內距、間距、圓角、邊框粗細
- 拖四角縮放（可鎖等比）、拖移位置
- Ctrl+Z 一步一步返回

## 🔒 隱私說明

這些工具**完全在你的瀏覽器本地運作**：

- 你上傳的圖片、貼上的 HTML 都只在你自己的瀏覽器裡處理
- 不會上傳到任何伺服器，作者也收不到任何資料
- 沒有後端、沒有資料庫、沒有追蹤

可以放心使用。
