🎯 Caveduck 內容對齊工具

這只是一個讓你更直覺對齊的小工具。
當你有一個素材框（例如掌機、相框、螢幕），卻一直沒辦法讓你的 CSS 內容剛好對齊那個框時——
① 放入你的素材圖　②　放入要對齊的 CSS 內容　③　拖四個角對齊框　④　產生程式碼。

① 上傳底圖

② 貼上你的內容 HTML

要丟什麼？
✓ 只丟「螢幕裡要顯示的內容」——例如你的指令卡 <div>...</div>。
✗ 不要丟最外層 max-width 容器、<img> 圖片行、position:absolute 或 matrix3d 定位框（這些工具會自動幫你弄）。
Caveduck 規則：全部用 style="..." 寫死，不要用 <style> 或 class，不要有連續空行。
